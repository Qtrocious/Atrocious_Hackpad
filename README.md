# BOM
- 9x Cherry MX Switches
- 9x Blank DSA Keycaps
- 9x SK6812 MINI Leds
- 9x Through-hole 1N4148 Diodes
- 3x M3x16 Bolt
- 3x M3 Heatset
- 1x EC11 Rotary encoder
- 1x OLED .91" SSD1306
- 1x XIAO RP2040
